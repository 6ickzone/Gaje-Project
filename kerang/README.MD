##  Features

### Dual Panel Explorer
- Browse folders & files side by side  
- Edit, rename, chmod, download, and delete  
- Create new file or folder directly  

### Command Execution
Supports multiple PHP methods:
- `shell_exec`
- `exec`
- `passthru`
- `system`
- `popen`

###  Upload Options
- Drag & Drop file upload  
- Upload via **wget**  
- Upload via **curl**

###  Server Info
Quick access to:
- `phpinfo()`  
- Basic system information  
- Current working directory
